# Archived Documentation

This directory contains archived documentation files that have been compressed to reduce repository size.

Contains audit reports and documentation from the JobPing codebase cleanup efforts in January 2026.

**To extract:**

## Archive Policy

Documentation files older than 30 days with no active references are automatically archived to maintain repository performance. Current documentation remains in the main `docs/` directory.
